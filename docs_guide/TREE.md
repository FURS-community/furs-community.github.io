# 开发的伙伴们，文件树如下图所示

```
furs-community.github.io/
├── main.html                  # 主页
├── pages/                      # 子页面目录
│   ├── news.html               # 新闻页
│   ├── about.html              # 关于我们
│   ├── join.html               # 加入我们
│   └── comingsoon.html         # 建设中页面
├── styles/                     # 样式文件目录
│   ├── main.css                # 主样式
│   └── pages/                  # 页面专属样式
│       ├── news.css            # 新闻页样式
│       └── about.css           # 关于页样式
├── scripts/                    # 脚本目录
│   └── main.js                 # 主脚本
├── assets/                     # 资源目录
│   ├── logo.ico                # 网站图标
│   ├── FURS.jpg                # 图片资源
│   └── FURS_VIDEO              # 视频资源
├── docs/                       # 项目文档
│   ├── COLLABORATION_GUIDE.md  # 协作指南
│   ├── TODO.md                 # 待办事项
│   ├── UPDATE.md               # 更新内容
│   └── TREE.md                 # 文件树
├── LICENSE                     # 许可证
└── README.md                   # 项目说明文件
```
